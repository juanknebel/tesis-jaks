git (opcional)
g++ (compilador)
cmake (automatizador para compilar)
mysql (mariadb-server mariadb-client) (base de datos)

libmysqlcppconn-dev libmysqlcppconn7 (conector para la bd mysql) --deprecated
qt5framewok (conector para la bd mysql)
